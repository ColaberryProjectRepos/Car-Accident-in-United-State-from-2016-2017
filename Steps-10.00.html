Create Dax measures and columns 10

DAX and Measures:

 

New Table:

Calendar = GENERATE (CALENDAR (DATE (2016, 02, 08), DATE (2017, 7,9)), VAR current Day = [Date] VAR day = DAY (currentDay) VAR month = MONTH (currentDay) VAR year = YEAR (current Day) RETURN   ROW ("day", day, "month", month, "year", year))


    Additional Columns:

1.    DayName = FORMAT ('Calendar' [day], "DDDD")

2.    MonthName = FORMAT (Calendar [Date], "MMMM")

3.    WeekDay = WEEKDAY ('Calendar' [day], 2)

4.    DayName = FORMAT('Calendar'[day], "DDDD")

 

 

UNITED STATE_Accident:

CALCULATED COLUMN:

1.   .Total Accident by traffic-signal = IF ('UNITED STATE_Accident'           [Traffic_Signal] = "True", 1, 0)

             Note: change the data type for [Traffic-Signal] to -Text- to Avoid getting Error through creating the calculated column.



MEASURES:

1.    .Accident At Traffic-Signal = SUM('UNITED STATE_Accident'[.Total  Acccident by traffic-signal])/COUNT('UNITED STATE_Accident'[ID])

 

2.    Total Traffi Acc. = SUM ('UNITED STATE_Accident'[.Total Acccident by traffic-signal])

 

 

3.    .Accident/Month = AVERAGEx(VALUES('Calendar'[Date].[Month]),'UNITED STATE_Accident'[.Total Traffi Acc.])

 

4.    Accident Prev.Month = CALCULATE ('UNITED STATE_Accident' [.Accident/Month], DATEADD ('Calendar' [Date], -1, MONTH))

 

 

5.    .MOM = DIVIDE([.Accident/Month]- [Accident Prev.Month], [Accident Prev.Month])

 

6.    Arrow MOM = SWITCH(TRUE(), [.MOM] < 0, UNICHAR(9660), UNICHAR(9650))

 

7.    ColorArrow Mom = IF ([.Arrow MOM] = UNICHAR (9660), "Green", "Red")

8.    .Total Accident = COUNT('UNITED STATE_Accident'[ID])



 the following is the Formatting Measures: Accident At Traffic-Signal and.MOM should be in Percentage
